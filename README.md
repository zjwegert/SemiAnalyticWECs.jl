# SemiAnalyticWECs.jl: A Julia-based solver for surface/submerged piezoelectric wave energy converters

[![](https://img.shields.io/badge/docs-stable-blue.svg)](https://zjwegert.github.io/SemiAnalyticWECs.jl/stable) [![](https://img.shields.io/badge/docs-dev-blue.svg)](https://zjwegert.github.io/SemiAnalyticWECs.jl/dev)

Codes related to the paper:
> Zachary J. Wegert, Ben Wilks, Ngamta Thamwattana, Vivien J. Challis, Santanu Koley, Michael H. Meylan (2025). "Wave energy conversion by floating and submerged piezoelectric bimorph plates"

## Installation
The package can be installed using the Julia package manager: type `]` in the Julia REPL to enter package mode, and install using
```julia
pkg> add https://github.com/zjwegert/SemiAnalyticWECs.jl
```

## Usage
The numerical experiments in the journal paper can be run via the `.jl` files in `/scripts`.
